This is a simple News App built using HTML, CSS, JavaScript, jQuery, and AJAX. It fetches real-time news from an API and displays them in a clean, responsive card layout. Users can browse news by category, search for specific topics, and click on a news card to view detailed information without reloading the page.

Features

Browse news by categories (Sports, Politics, Finance)

Search news using keywords

Click on a news card to view full details

Responsive and modern UI design

Uses jQuery AJAX to fetch data from API
